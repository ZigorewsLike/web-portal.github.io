Блоги

  • Блог HTML академии. Много полезного материала от отличного ресурса дистанционного обучения.
  • Йоксель. Блог Юли Бухваловой, верстальщика Livejournal. Качественные статьи по CSS + множество полезных тематических ссылок, больше чем на странице, которую Вы сейчас читаете.
  • FAQ на блоге Павла Радькова. Советы по дизайну, вёрстке, организации труда и веб-разработке в целом
  • CSS-Live. "Жизнь во фронтедне". Тонны материала.
  • Пепелсбей.net. Блог Вадима Макеева. Не пишет в этот блог с 2014 года, но всё-равно есть что почитать. Автор крутого движка для презентаций в web формате.
  • Блог Юрия Матюхина. В заметках есть неплохие ссылки.
  • Блог Нетологии
  • Блог Антона Филиппова

Справочники

  • WebReference. Справочник номер 1 в рунете по HTML, CSS и смежным технологиям, вторая версия HTMLBook.
  • HTML book. Устаревшая версия WebReference. Только HTML и CSS Тоже ничего.
  • Can I use. Таблица поддержки браузерами современных веб технологий.
  • Developer Documents. Дока на все случаи жизни.

Ресурсы

  • CSS Tricks. Just must have. It used to be, believe it or not, primarily about CSS! Решения всех типовых задач CSS и не только. (На английском).
  • Документация от Mozilla. Web технологии, обучение, инструменты разработчика и другие материалы.
  • Учебник по JavaScript. Лучший учебник по JavaScript.
  • Smashing magazine. Журнал про веб технологии (на английском).

Интересное

  • Путь веб разработчика 2017. Три возможных пути становления веб-разработчика:frontend, backend и DevOps. С переводом на русский язык от Библиотеки программиста или Нетологии
  • JavaScript Правильный путь
  • CSS Grid by Example. Обучение, примеры, шаблоны CSS Grid.
  • Карта развития веб-разработчика
  • Шаблоны проектирования для людей. Примеры про двери, бургеры и т.п. на PHP. Есть переводной вариант от Mail.ru на Хабре.
